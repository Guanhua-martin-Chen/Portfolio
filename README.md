### 1. Cross-Dataset Lottery Ticket Hypothesis
ðŸ”— https://github.com/TonyZhu2002/Cross-Dataset-LTH  

A research-oriented project exploring whether **sequential pruning across multiple datasets** can produce more **universal sparse subnetworks** under the Lottery Ticket Hypothesis (LTH).  
Implemented a full **cross-dataset iterative magnitude pruning (IMP)** pipeline using MNIST and Fashion-MNIST, and evaluated transfer performance on unseen **grayscale CIFAR-10**.  
Built a reproducible PyTorch framework with checkpointed pruning stages, mask-freezing logic, and comprehensive evaluation including accuracy curves, confusion matrices, and per-class analysis.  
Results show cross-dataset tickets outperform single-dataset and random sparse baselines at high sparsity, suggesting improved transferability.
