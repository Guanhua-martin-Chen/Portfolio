## Featured Projects

### 1. [Immune Cell Composition Analysis](https://github.com/Guanhua-martin-Chen/Transcriptomic-Signatures-of-Host-Cytokine-Responses-Reveal-Determinants-of-virus-Infection-Outcomes)
A transcriptomics-based analysis of blood immune-cell composition across infection stages.  
Built an end-to-end pipeline: transcript → gene aggregation, xCell enrichment (blood cell types), metadata integration, Wilcoxon & CLR-based statistical testing, and final summary visualization.  
Produced PI-requested comparative barplots and DSSO-based trends to highlight immune shifts in infected cohorts.
